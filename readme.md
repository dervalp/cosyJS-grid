##API

###Create

arguments can be : ```foundation|boostrap|boostrap-fluid``` or an object:

```javascript
{
    prefix: "large-",
    row: "row",
    extra: "columns",
    mobilePrefix: "small-",
    offsetPrefix: "large-offset-",
    offsetMobilePrefix: "small-offset-",
    centered: "large-centered",
    mobileCentered: "small-centered",
}
```
